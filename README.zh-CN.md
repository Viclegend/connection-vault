# 🛡️ 连线信息库 (Connection Vault)

[繁體中文](README.md) | [简体中文](README.zh-CN.md) | [English](README.en.md)

这是一个极致轻量化、基于 Docker 容器运行的 Web 应用程序。专为系统集成 (SI) 团队、企业 MIS，以及拥有众多 Hyper-V 或 Kubernetes (K8s) 节点的 Home Lab 玩家设计。帮助您告别散落各处的密码表与 Excel，集中且安全地管理不同分组、机房或虚拟机环境的连线信息。

## 🌟 系统特色

* 🔄 **批量导入与备份：** 支持标准 CSV 格式一键导入。遇到新分组会自动建档；遇到重复设备则自动覆盖更新账号密码。
* 🔍 **智能全局搜索：** 实时筛选分组名称、设备名称、IP 或账号，寻找节点零延迟。
* 🔒 **安全防护：** 登录验证、敏感密码默认遮罩（支持一键显示/隐藏），并具备防误删机制。
* 🌗 **现代化 UI：** 干净利落的手风琴折叠界面，内置日/夜间主题切换。

## 🐳 快速部署指南

本系统依赖 Docker 运行，请确保您的服务器已安装 Docker Compose。

**1. 下载项目**
```bash
git clone [https://github.com/Viclegend/connection-vault.git](https://github.com/Viclegend/connection-vault.git)
cd connection-vault
```

**2. 环境变量设定**
复制模板文件，并设定您的对外通讯端口与管理员登录账号密码：
```bash
cp .env.example .env
```

**3. 一键启动服务**
```bash
docker compose up -d
```
🎉 **开始使用：** 打开浏览器前往 `http://<服务器IP>:<设定的Port>` 即可登录！所有的数据将会安全地持久化存储于 `./data` 目录中。

## 🔄 数据导入规范

若要使用系统内的“导入”功能，请严格遵守以下格式：
1. **编码与格式：** 仅支持带有 **UTF-8 BOM** 编码的 `.csv` 文件（建议直接从系统下载“范例文件”进行编辑）。
2. **覆盖机制：** 若导入的数据中包含系统已存在的“分组+设备”，系统将自动以新 CSV 文件中的账号密码进行覆盖更新。
